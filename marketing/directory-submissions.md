## Submission etiquette

- One PR per awesome-list, with a clear "fits under X section because Y" justification.
- Never submit to a directory you wouldn't actually use.
- Don't paste the same blurb everywhere — at least vary the framing per surface.
- For HN / Lobste.rs / Reddit "Show" posts: comment first with your own honest tradeoffs (3 sentences). High signal-to-self-promo ratio = no flag.
- For directory submissions that ask for an email: use a project-specific alias if you have one, or your main email — never burn a fresh inbox.
